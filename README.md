# ParkNow - Εφαρμογή Κράτησης Χώρων Στάθμευσης

## Περιγραφή
Το ParkNow είναι μια εφαρμογή για κινητά που επιτρέπει στους χρήστες να αναζητούν, να κρατούν και να διαχειρίζονται χώρους στάθμευσης. Υποστηρίζει δύο ρόλους χρηστών:
- **Driver (Οδηγός)**: Αναζήτηση και κράτηση χώρων στάθμευσης
- **Host (Ιδιοκτήτης)**: Προσθήκη και διαχείριση χώρων στάθμευσης

---

## 📋 Απαιτήσεις Συστήματος

- **Flutter SDK**: 3.0 ή νεότερο
- **Dart SDK**: 2.17 ή νεότερο
- **Android**: API Level 21 (Android 5.0) ή νεότερο

## 🚀 Εγκατάσταση

### Μέθοδος 1: Εγκατάσταση APK (Απλή - Συνιστάται)

1. Κατεβάστε το αρχείο `parknow-release.apk`

3. Ανοίξτε το APK και πατήστε **"Εγκατάσταση"**

1. Εγκαταστήστε το Flutter: https://flutter.dev/docs/get-started/install
2. Αποσυμπιέστε το `flutter_application_1.zip`
3. Ανοίξτε terminal στον φάκελο του project
4. Εκτελέστε:
   ```bash
   flutter pub get
   flutter run
   ```

## 👤 Οδηγίες Χρήσης

### Πρώτη Εκκίνηση

1. **Εγγραφή**:
   - Πατήστε "Εγγραφή"
   - Συμπληρώστε: Όνομα, Email, Κωδικό
   - Επιλέξτε ρόλο: **Driver** (για αναζήτηση χώρων) ή **Host** (για προσθήκη χώρων)

2. **Σύνδεση** (για επόμενες φορές):
   - Email: Οποιοδήποτε (π.χ. `test@test.com`)
   - Κωδικός: Οποιοσδήποτε
   - *Σημείωση: Η εφαρμογή δέχεται οποιαδήποτε διαπιστευτήρια (Dev Mode)*

---

### Λειτουργίες Driver (Οδηγός)

#### 1. Αναζήτηση Χώρων
- **Αρχική**: Πατήστε το search bar "Πού θέλεις να παρκάρεις;"
- **Αναζήτηση**: Πληκτρολογήστε περιοχή (π.χ. "Κολωνάκι", "Σύνταγμα")
- **Φίλτρα**: Πατήστε το εικονίδιο φίλτρων για να επιλέξετε:
  - Εύρος τιμής
  - Παροχές (Στεγασμένο, Κάμερες, EV Charging, κλπ)

#### 2. Κράτηση Χώρου
1. Επιλέξτε έναν χώρο από τα αποτελέσματα
2. Δείτε λεπτομέρειες (τιμή, αξιολογήσεις, χάρτης)
3. Πατήστε **"Book"**
4. Επιλέξτε:
   - Ημερομηνία & Ώρα
   - Διάρκεια (ώρες ή ημέρες)
5. Επιβεβαιώστε και πληρώστε
6. Λάβετε **PIN κωδικό** για πρόσβαση

#### 3. Οι Κρατήσεις μου
- **Προφίλ** → **"Οι Κρατήσεις μου"**
- Δείτε:
  - **Ενεργές**: Κρατήσεις με PIN
  - **Ιστορικό**: Παλιές κρατήσεις
- Πατήστε μια κράτηση για να δείτε τον PIN

---

### Λειτουργίες Host (Ιδιοκτήτης)

#### 1. Προσθήκη Χώρου Στάθμευσης
1. **Dashboard** → **"Οι Χώροι μου"** → **"+"**
2. Συμπληρώστε:
   - Τίτλος (π.χ. "Γκαράζ Κολωνακίου")
   - Διεύθυνση
   - Τιμή ανά ώρα
3. Επιλέξτε μέθοδο πρόσβασης
4. Πατήστε **"Δημοσίευση"**

#### 2. Στατιστικά & Έσοδα
- **Dashboard** → **"Στατιστικά"**
- Δείτε:
  - Έσοδα μήνα (πραγματικά από κρατήσεις)
  - Αριθμό κρατήσεων
  - Μοναδικούς πελάτες
  - Μέσο όρο ανά κράτηση

#### 3. Εναλλαγή Ρόλου
- **Ρυθμίσεις** → **"Εναλλαγή σε Driver/Host"**
- Αλλάξτε ρόλο ανά πάσα στιγμή

## 🎯 Σενάρια Χρήσης

### Σενάριο 1: Κράτηση Χώρου
1. Συνδεθείτε ως **Driver**
2. Αναζητήστε "Κολωνάκι"
3. Επιλέξτε "Κολωνάκι Parking 1"
4. Κάντε κράτηση για 2 ώρες
5. Πληρώστε
6. Δείτε τον PIN στις κρατήσεις σας

### Σενάριο 2: Προσθήκη Χώρου & Παρακολούθηση Εσόδων
1. Συνδεθείτε ως **Host**
2. Προσθέστε νέο χώρο
3. Αλλάξτε σε **Driver**
4. Κάντε κράτηση στον χώρο σας
5. Αλλάξτε πίσω σε **Host**
6. Δείτε τα έσοδα στα Στατιστικά

## 💾 Διαχείριση Δεδομένων

### Τοπική Αποθήκευση (In-Memory)
Η εφαρμογή αποθηκεύει δεδομένα στη μνήμη κατά τη διάρκεια της συνεδρίας:
- Χρήστες (email, όνομα, ρόλος)
- Κρατήσεις (με PIN, ώρες, τιμές)
- Χώροι στάθμευσης (με ιδιοκτήτη)

**Σημείωση**: Τα δεδομένα διατηρούνται μόνο όσο η εφαρμογή είναι ανοιχτή. Σε επανεκκίνηση, επαναφέρονται τα αρχικά mock δεδομένα.

### Ροή Δεδομένων
- **Κράτηση → Προφίλ**: Οι κρατήσεις εμφανίζονται αυτόματα στο προφίλ
- **Κράτηση → Host Stats**: Τα έσοδα υπολογίζονται αυτόματα
- **Νέος Χώρος → Χάρτης**: Εμφανίζεται αμέσως στα αποτελέσματα

## 📝 Διαφορές από το Prototype

1. **Πραγματική Λογική Κρατήσεων**: Αντί για mock, οι κρατήσεις συνδέονται με χρήστες
2. **PIN Codes**: Κάθε κράτηση έχει μοναδικό PIN
3. **Host Dashboard**: Πραγματικά έσοδα από κρατήσεις
4. **Profile**: Συνδεδεμένο με πραγματικά δεδομένα χρήστη
5. **Spot Ownership**: Οι χώροι συνδέονται με τον host που τους δημιούργησε
