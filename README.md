ParkNow is a cross-platform mobile and web application developed with Flutter, designed to help users quickly find, manage, and navigate parking options in urban environments.

The goal of the project is to provide a simple and efficient parking experience by combining real-time availability, location services, and user-friendly navigation in one unified platform.  
The application targets drivers who want to reduce time spent searching for parking and improve their overall urban mobility experience.

## Features (Planned / In Progress)

- Real-time parking availability
- Interactive map with navigation support
- Cross-platform support (Android, iOS, Web, Windows, macOS, Linux)
- User accounts and parking history
- Smart suggestions based on location and demand

## Technology Stack

- Flutter / Dart
- Google Maps API (planned)
- Cross-platform deployment (mobile, web, desktop)
