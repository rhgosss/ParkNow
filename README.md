
Σύνοψη Εφαρμογής
Το ParkNow είναι μια εφαρμογή διαμοιρασμού θέσεων στάθμευσης (Parking Sharing Economy), υλοποιημένη σε Flutter. Επιτρέπει σε ιδιοκτήτες θέσεων (Hosts) να νοικιάζουν τις θέσεις τους και σε οδηγούς (Drivers) να τις βρίσκουν στον χάρτη, να τις κλείνουν και να πληρώνουν εύκολα.

α) Οδηγίες Εγκατάστασης & Χρήσης
Η εφαρμογή έχει σχεδιαστεί για να είναι Plug & Play. Δεν απαιτείται καμία τοπική εγκατάσταση βάσης δεδομένων, καθώς χρησιμοποιεί Firebase (Cloud).

Τρόπος 1: Εγκατάσταση μέσω APK (Προτείνεται για απλούς χρήστες)
Ο πιο εύκολος τρόπος να τρέξετε την εφαρμογή είναι εγκαθιστώντας το έτοιμο αρχείο .apk σε μια Android συσκευή.

Κατεβάστε το αρχείο .apk από τον παρακάτω σύνδεσμο: 👉 Σύνδεσμος λήψης APK (Google Drive)

Μεταφέρετε το αρχείο στο Android κινητό σας (ή Emulator).

Πατήστε πάνω του και επιλέξτε "Εγκατάσταση" (Install). Ενδέχεται να χρειαστεί να επιτρέψετε την εγκατάσταση από άγνωστες πηγές.

Βεβαιωθείτε ότι η συσκευή έχει ενεργή σύνδεση στο Internet (απαραίτητο για Login & Χάρτες).

Τρόπος 2: Εκτέλεση από τον Κώδικα (Για Developers/Reviewers)
Αν επιθυμείτε να τρέξετε τον κώδικα μέσω Flutter/IDE:

Κάντε Clone το repository: git clone https://github.com/rhgosss/ParkNow.git

Ανοίξτε το project φάκελο σε VS Code ή Android Studio.

Ανοίξτε τερματικό και τρέξτε: flutter pub get

Συνδέστε μια συσκευή ή ανοίξτε έναν Emulator.

Τρέξτε την εντολή: flutter run

Σημαντική Σημείωση: Δεν απαιτείται καμία ρύθμιση firewall ή βάσης δεδομένων. Όλα τα services (Auth, Database, Storage) φιλοξενούνται στο Firebase και είναι ενεργά.

β) Τεχνικές Απαιτήσεις (Android SDK & Environment)
Για την ορθή λειτουργία της εφαρμογής απαιτούνται τα εξής:

Android SDK Version:

Min SDK: 21 (Android 5.0 Lollipop)

Target SDK: 34 (Android 14)

Google Play Services: Η συσκευή ή ο Emulator ΠΡΕΠΕΙ να διαθέτει Google Play Services (Google APIs image), καθώς η εφαρμογή χρησιμοποιεί Google Maps.

Σε Emulator: Επιλέξτε image που έχει το εικονίδιο του Play Store δίπλα ή γράφει "Google APIs".
